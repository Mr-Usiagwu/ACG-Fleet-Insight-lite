**✈️ ACG Fleet Insight Lite**
An Aviation Asset Management & Predictive Maintenance Dashboard

📌 Project Overview
The ACG Fleet Insight Lite is a high-level Decision Support System (DSS) designed for aircraft lessors, fleet managers, and airline operators. It bridges the gap between technical maintenance data and financial exposure, allowing users to visualize fleet health and financial risk in real-time.

🎯 Business Value
Operational Efficiency: Real-time tracking of engine cycles and maintenance intervals.

Financial Oversight: Calculation of "Financial Exposure" based on fleet-wide maintenance status.

Proactive Maintenance: Predictive indicators for "High Risk" assets requiring immediate attention.

🛠️ Technical Stack
Language: Python 3.13

Framework: Streamlit (Web UI)

Data Handling: Pandas & NumPy

Visualization: Plotly (Interactive Charts)

Deployment: Streamlit Cloud / GitHub

🚀 Key Features
Multi-Role View: * Executive: High-level fleet health and financial totals.

Finance: Detailed maintenance reserve and exposure tracking.

Operations: Granular engine-by-engine cycle tracking and maintenance actions.

Dynamic Simulation: An engine.py backend that simulates real-world flight cycle accumulation.

Interactive Maintenance: Users can simulate "Performing Maintenance" to see real-time drops in financial risk scores.
